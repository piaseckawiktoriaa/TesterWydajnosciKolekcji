# 📊 Tester wydajności kolekcji w Java

## 📌 Opis projektu

Aplikacja konsolowa napisana w języku Java służąca do testowania i porównywania wydajności różnych typów kolekcji.

Program umożliwia analizę działania takich struktur jak:

* ArrayList
* LinkedList
* HashSet
* TreeSet

Użytkownik może konfigurować testy za pomocą interfejsu CLI (konsola), wybierając typ danych, liczbę elementów oraz rodzaj kolekcji.

---

## ⚙️ Funkcjonalności

* generowanie danych:

  * Integer
  * Double
  * obiekty (Person, MyColor, Ksiazka, Samochod)

* wybór liczby elementów (100 – 10 000 lub własna)

* pomiar czasu:

  * dodawania elementów
  * wyszukiwania (contains)
  * usuwania (remove)

* odczyt elementu po indeksie (dla list)

* zapis wyników do pliku CSV

* prezentacja wyników w konsoli

---

## 🛠️ Technologie

* Java
* Java Collections Framework
* Stream API
* Enumy i interfejsy
* Operacje na plikach (FileWriter)
* CLI (aplikacja konsolowa)

---

## ▶️ Jak uruchomić

1. Otwórz projekt w IntelliJ IDEA lub innym IDE
2. Skompiluj plik `.java`
3. Uruchom klasę główną
4. Postępuj zgodnie z instrukcjami w konsoli
